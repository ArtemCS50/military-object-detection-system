# Military Object Detection - React Frontend
Modern React-based web interface for the Military Object Detection System.
## Overview
A responsive single-page application (SPA) that provides an intuitive interface for uploading images and videos, viewing real-time processing results, and managing user authentication.
## Features
- 🔐 **User Authentication** - Register, login, and protected routes
- 📤 **File Upload** - Drag-and-drop image and video upload
- 🎥 **Real-time Processing** - Live video processing with WebSocket streaming
- 📊 **Analytics Dashboard** - View object detection statistics
- 🎨 **Responsive Design** - Bootstrap-based responsive UI
- 🔄 **State Management** - React Context API for global state
## Tech Stack
- **React 18.3.1** - UI framework
- **React Router DOM v7** - Client-side routing
- **React Bootstrap 5.3.3** - UI components
- **Axios** - HTTP client for API requests
- **Socket.io Client** - Real-time bidirectional communication
- **React Context API** - State management
## Available Scripts
### \`npm start\`
Runs the app in development mode at \`http://localhost:3000\`
### \`npm test\`
Launches the test runner
### \`npm run build\`
Builds the app for production to the \`build\` folder
